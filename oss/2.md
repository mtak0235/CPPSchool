1. git commit 에 대한 설명 중 가장 적합한 것을 선택하시오.

- [x] 작업한 파일을 주 branch에 올리기
- [ ] 주 branch에서 작업한 파일을 내려받기
- [ ] 주 branch에서 새로운 branch로 갈라치기
- [ ] 갈라진 branch를 주 branch와 합하기
